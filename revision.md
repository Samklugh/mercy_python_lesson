# 🐍 Python Beginner Exercises

## 🟢 Beginner Level

### 1. Print a Message
Write a program that prints "Hello, World!"  
**Hint:** Use the `print()` function.

---

### 2. User Input
Ask the user for their name and print a greeting.  
**Hint:** Use `input()` and string concatenation.

---

### 3. Simple Addition
Take two numbers as input and print their sum.  
**Hint:** Convert input to integers using `int()`.

---

### 4. Even or Odd
Check if a number is even or odd.  
**Hint:** Use the modulus operator `%`.

---

### 5. Positive, Negative, or Zero
Determine whether a number is positive, negative, or zero.  
**Hint:** Use `if`, `elif`, and `else`.

---

## 🟡 Beginner–Intermediate Level

### 6. Largest of Three Numbers
Find the largest of three numbers entered by the user.  
**Hint:** Use multiple `if` statements or `max()`.

---

### 7. Multiplication Table
Print the multiplication table of a number (1–10).  
**Hint:** Use a `for` loop.

---

### 8. Sum of Numbers
Calculate the sum of numbers from 1 to `n`.  
**Hint:** Use a loop or `sum()`.

---

### 9. Factorial Calculator
Compute the factorial of a number.  
**Hint:** Use a loop or recursion.

---

### 10. Reverse a String
Reverse a string entered by the user.  
**Hint:** Use slicing `[::-1]`.

---

## 🟠 Intermediate Basics

### 11. Count Vowels
Count the number of vowels in a string.  
**Hint:** Loop through the string and check vowels.

---

### 12. Palindrome Check
Check if a word is a palindrome.  
**Hint:** Compare the string with its reverse.

---

### 13. List Sum
Find the sum of all elements in a list.  
**Hint:** Use a loop or `sum()`.

---

### 14. Find Maximum in List
Find the largest number in a list.  
**Hint:** Use `max()` or a loop.

---

### 15. Remove Duplicates
Remove duplicates from a list.  
**Hint:** Convert list to a `set`.

---

## 🔵 Practice with Loops & Logic

### 16. Fibonacci Sequence
Print the first `n` Fibonacci numbers.  
**Hint:** Use a loop and update variables.

---

### 17. Prime Number Check
Check if a number is prime.  
**Hint:** Loop from 2 to `n-1`.

---

### 18. Number Guessing Game
Create a game where the user guesses a random number.  
**Hint:** Use `random.randint()`.

---

### 19. Simple Calculator
Build a calculator (+, -, *, /) using functions.  
**Hint:** Define separate functions for each operation.

---

### 20. Count Letters in Word
Count how many times each letter appears in a word.  
**Hint:** Use a dictionary and loop through the word.
